# 🤝 KeenKeeper

## 📖 Short Description

KeenKeeper is a personal relationship management dashboard designed to help you browse, tend, and nurture the relationships that matter most. It allows users to keep track of their friends, log interactions, set relationship goals, and view visual analytics of their communication habits.

## 🚀 Technologies Used

- **Frontend Library:** React (with Vite)
- **Styling:** Tailwind CSS
- **Routing:** React Router DOM
- **Data Visualization:** Recharts
- **Icons:** Lucide React
- **Notifications:** React Hot Toast

## ✨ 3 Key Features

1. **Interactive Timeline & Check-ins:** Easily log your interactions (Call, Text, Video) with friends and view a complete, filterable history of your communications.
2. **Visual Friendship Analytics:** Understand your communication habits through interactive Pie Charts that show the exact breakdown of your interaction types.
3. **Fully Responsive Dashboard:** A beautifully designed, seamless user interface that works perfectly across mobile, tablet, and desktop devices without breaking the layout.
